## Features
- **Prefix**: .
- **.hi**: Returns a Simple Reply Response.
- **.ping**: Returns a simple Ping Response, with Latency.
- **Events**: on_ready,on_disconnect,on_reconnect,on_message Included.
- **Embeds**: .sendEmbed and .embed use the redEmbed API, to Send Embeds.
- **Extra**: Multiple Functions Have Been added, to make it more Visually Appealing.
